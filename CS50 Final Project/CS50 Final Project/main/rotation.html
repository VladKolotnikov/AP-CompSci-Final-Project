Rotational Physics

Taking a little break from the world of vectors and froces, a brief introduction in rotational physics, and the last thing I had time to start for this project. Rotational physics is interesting, because while it still is more complicated than linear physics, just like in the real world, it has one simplification in terms of code. Rotational values are all scalar quantaties, which means no more fuss with vectors, we are back to first grade math. Nice!

Rotational Kinematics

Kinematics is a good place to start, because it is almost the same in its both linear and rotational form, minus the vector stuff. Angular position is by how much an object has already rotated, angular velocity is how many radiants it rotates per second, and angular acceleration modifies velocity. In other words, it is the same Euler Integration logic. A proof of concept here, I set the magnitute of acceleration to depend on how far from the rectangle the mouse is.
